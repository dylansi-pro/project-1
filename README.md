# Koko Café - Brand Website

Ce projet consiste en la création d'un site vitrine complet pour **Koko Café**, un café de spécialité fictif proposant des pâtisseries artisanales et des brunchs dans une ambiance chaleureuse.

Il a été réalisé dans le cadre du **Project 1: Brand Website**.

## Fonctionnalités du projet

- **Structure fixe de 960px** centrée avec ombres portées, optimisée pour Firefox et Chrome.
- **Menu de navigation fixe** supérieur intégrant des menus déroulants (_Dropdown_).
- **Navigation fluide** sur un minimum de 5 pages interconnectées (`Home`, `About`, `Menu`, `Gallery`, `Contact`).
- **Design moderne et épuré** conçu intégralement en HTML5 et CSS3 pur (sans frameworks externes).
- **Formulaire de contact sémantique** utilisant la méthode `POST` et relié au serveur de test de l'institut.
- **Intégration de polices web** (Google Fonts : _Playfair Display_ et _Roboto_).
- **Mises en page modernes** exploitant CSS Flexbox (sections, staff) et CSS Grid (galerie de photos).

## Structure des fichiers

```text
project-1/
├── css/
│   └── style.css       # Fichier CSS externe unique contenant tous les styles
├── images/             # Dossier regroupant toutes les images libres de droits
├── index.html          # Page d'accueil (Home)
├── about.html          # Page de présentation de l'équipe (About us / Staff)
├── menu.html           # Carte du café, pâtisseries et brunchs
├── gallery.html        # Galerie photo interactive (Grille CSS)
└── contact.html        # Formulaire de contact fonctionnel (POST)
```
